Manual — SIPOC Builder v7.0

Flujo de trabajo

  1. Inicio — Elige captura guiada o carga del contrato JSON del LMP.
  2. Datos del proceso — Registra o normaliza el proceso, carriles, SIPOC y actividades.
  3. Diagram Studio — Valida el contrato y construye automáticamente nodos y conexiones.
  4. Validación — Comprueba IDs, relaciones, preservación y estructura draw.io.
  5. Salida — Incorpora encabezado y notas y genera el XML final.

Importar en draw.io

  1. Abre app.diagrams.net
  2. Ve a Extras → Edit Diagram
  3. Borra el contenido, pega el XML y haz clic en Cerrar
  4. Usa Ctrl+Shift+H para ajustar la vista

Preguntas de entrevista para el LMP

Del proceso:

  • ¿Cuál es el código y nombre oficial del proceso?
  • ¿Cuál es la finalidad en una oración?
  • ¿Quiénes son los proveedores externos (Suppliers)?
  • ¿Qué entradas externas recibe el proceso (Inputs)?
  • ¿Qué salidas genera hacia el exterior (Outputs)?
  • ¿Quiénes son los receptores de esas salidas (Customers)?
  • ¿Qué actores o roles participan internamente?
  • ¿Qué documentos guían las actividades?
  • ¿Hay notas aclaratorias o excepciones?

Por actividad:

  • ¿Cuál es el nombre? ¿Es operación, decisión o control?
  • ¿Quién la ejecuta? ¿Qué documento usa?
  • ¿Qué actividad viene antes? ¿Qué actividad sigue?
  • ¿Es la primera actividad del flujo? ¿Es la última?
