1. Modo de trabajo

Selecciona si se mejorará un documento existente o si se construirá desde cero.

Mejorar documento existenteEl usuario carga Word, Excel o texto; la salida conserva el mismo formato.
Crear documento desde ceroEl usuario define contexto y elige salida Word o Excel.

2. Configuración común

Estos campos alimentan el bloque maestro del prompt. La IA actuará como experta en industria, normativa, metodología y proceso.

3. Normativas y metodologías

El usuario puede seleccionar las existentes o agregar nuevas. El prompt no inventará vigencias: pedirá declarar limitaciones cuando no haya certeza.
